Privacy Policy

HolyMatch uses Apple, Firebase, and RevenueCat services for authentication, analytics, purchases, and subscriptions. Ads are provided via Google Mobile Ads (AdMob).

Data We Process

  • Account/Identifiers: Firebase User ID, device identifiers (for ads/analytics).
  • Purchases: Subscription status and transactions via Apple & RevenueCat.
  • Usage/Diagnostics: Basic app interactions and crash logs to improve stability.

Tracking / Ads

We may request permission to track (AppTrackingTransparency) for ad personalization and measurement. You can deny this and continue to use the app.

Security

All network communication uses HTTPS/TLS. We do not sell personal data.
